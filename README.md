### Сначала склонируйте проект целиком:

```shell
git clone https://github.com/koryaginsergey123/agona.git
```

### Далее зайдите в нужный проект, например:

```shell
cd lesson-20-09-23
```

### запустите устанощик пакетов npm/yarn

```shell
yarn
```
или
```shell
npm i
```
